# Icons Placeholder

Для production нужны иконки:
- icon16.png (16x16)
- icon48.png (48x48)
- icon128.png (128x128)

Можешь создать их в любом редакторе (Figma, Photoshop, GIMP) или сгенерировать через AI (DALL-E, Midjourney).

Для MVP можно использовать простые placeholder иконки или вообще убрать из manifest.json временно (расширение будет работать, просто без иконки).

Если хочешь быстрый placeholder:

1. Создай квадратную картинку 128x128 с текстом "JM" (Job Match)
2. Resize до 48x48 и 16x16
3. Положи в `extension/public/icons/`

Или используй онлайн генератор: https://realfavicongenerator.net/
